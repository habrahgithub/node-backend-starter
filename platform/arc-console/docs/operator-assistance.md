# Operator Assistance

## What the Copilot Provides
- Interpretation of intelligence risks into clear operator guidance
- Diagnostic playbooks for unstable services
- Repository cleanup guidance with governance context
- Workflow suggestions mapped to risk domains
- Prioritized alert feed for urgent issues

## Usage Flow
1. Open `/assistant` for high-level copilot guidance.
2. Open `/diagnostics` for service troubleshooting sequences.
3. Open `/repo-advisor` for repo cleanup recommendations.
4. Open `/alerts` for prioritized platform notifications.
5. Approve only the workflows/actions you explicitly want to run.

## Approval Constraint
All suggestions are advisory and include:
- evidence
- confidence
- operator approval requirement

No suggestion executes automatically.

## Recommended Operator Practice
- Start with urgent alerts.
- Validate evidence sources.
- Run approved workflow simulations.
- Record governance decisions and outcomes.
