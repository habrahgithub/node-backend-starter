# UI QA Checklist

## Structure

- Semantic landmarks are present.
- Heading hierarchy is valid and intentional.
- Interactive controls use correct elements.

## Responsiveness

- Layout works at 360px, 768px, 1024px, 1440px.
- Text does not overflow or clip.
- Tap targets are usable on mobile.

## Accessibility

- Focus styles are visible.
- Inputs have labels and clear validation errors.
- Color contrast is adequate for primary text and controls.

## Behavior

- Loading and error states render correctly.
- Empty states have clear next action.
- Animations do not block usability.
