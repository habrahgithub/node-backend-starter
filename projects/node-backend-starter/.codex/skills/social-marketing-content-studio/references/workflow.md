# Production Workflow

## 1) Campaign Input

- Objective (awareness / leads / conversion)
- Audience
- Platform list
- CTA and landing URL

## 2) Creative Build

- Generate base image(s)
- Produce alternates for hooks/thumbnails
- Capture browser demo
- Generate narration

## 3) Composition

- Add headline overlay
- Add subtitles or key bullets
- Mix narration with demo footage

## 4) Export

- 1:1, 4:5, 9:16 variants
- Filename format: `<campaign>-<platform>-<ratio>-vN.<ext>`

## 5) QA

- Check readability and safe zones
- Check branding consistency
- Check links/CTA correctness
