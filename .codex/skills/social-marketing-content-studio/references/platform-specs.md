# Platform Specs Contract

Use this table as the default export contract.

| Platform | Preferred Ratio(s) | Duration Target | Caption Length |
| --- | --- | --- | --- |
| LinkedIn | 1:1, 4:5, 9:16 (video) | < 90 sec | 300-1200 chars |
| Instagram Reel/Story | 9:16 | < 60 sec | Short caption + hashtags |
| Instagram Feed | 1:1, 4:5 | N/A for stills | 125-500 chars |
| X | 1:1 (preferred), 16:9 | < 120 sec | <= 280 chars |
| YouTube Shorts | 9:16 | < 60 sec | 60-200 chars + hashtags |

## Pixel Targets

1. 1:1 -> 1080x1080
2. 4:5 -> 1080x1350
3. 9:16 -> 1080x1920
4. 16:9 -> 1920x1080

## Readability + Safe Zone

1. Keep critical text/logo inside center 80% frame.
2. Avoid top/bottom UI-clutter zones on vertical formats.
3. Minimum on-screen text size should remain legible on ~6.1-inch displays.
