# Hook Library

Use a short, direct hook in the first 2 seconds.

## Payroll / Compliance Hooks

1. "Still generating WPS files manually?"
2. "One formatting error can delay payroll."
3. "UAE SMEs do not need complex payroll software."
4. "Stop losing hours to repetitive payroll prep."

## Product Value Hooks

1. "Create payroll-ready files in minutes."
2. "From data to compliant output with fewer clicks."
3. "Cut prep time, keep control."
4. "Built for fast, reliable HR operations."

## Selection Rules

1. Match hook to campaign goal:
   - Awareness: pain-first hook
   - Conversion: outcome-first hook
2. Keep hook under 12 words when possible.
3. Avoid hype claims that cannot be validated.
