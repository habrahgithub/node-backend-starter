# Context

This repository is a workspace root that aggregates multiple project directories and supporting tools.
Governance references live under `docs/`.

Operating rules:
- Keep diffs small and tightly scoped.
- Do not paste logs into docs or issues; summarize and link to paths.
- Do not duplicate specs; update the canonical doc instead.
- Do not modify existing business logic unless explicitly instructed.
- Record notable decisions in `docs/DECISIONS.md`.
